\chapter{Existing File Example}

Using the file\_exists\_callback in the OptArgs dictionary and passing this dictionary to the MultiFormat constructor (directly or using the create\_mf function) allows us to handle an exiting file in any way we want. The default behaviour if no callback is provided is to refuse to overwrite existing files.
