\textit{With new\_paragraph we can start a second paragraph with the first sentence in italic.} Use add\_text to add text without italic to the second paragraph.\textbf{ Use add\_text to add text in bold to the second paragraph.}\textit{\textbf{ Use add\_text to add text in italic and bold to the second paragraph.}}
