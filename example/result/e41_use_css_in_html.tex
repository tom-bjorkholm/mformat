\chapter{CSS und Sprache in HTML}

Dieses Beispiel zeigt, wie Sie eine CSS-Datei und die Dokumentensprache (lang) mit mformat setzen. Übergeben Sie OptArgs mit "css\_file" und "lang" an create\_mf, wenn das Format HTML ist.

Die CSS-Datei liegt unter example/css/ und wird per relativem Pfad eingebunden (für lokale Anzeige). Die Ausgabe ist auf Deutsch; lang="de" steht im erzeugten \textless{}html\textgreater{}-Tag.
