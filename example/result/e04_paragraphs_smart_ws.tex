With new\_paragraph we can start a paragraph. Thanks to smart whitespace handling, we do not need to add whitespace between text fragments from different calls to add\_text or new\_paragraph calls. If we have extra whitespace, it will be consolidated into a single space.

With new\_paragraph we can start another paragraph.  With smart\_ws=False the whitespace between text  fragments will be preserved.So we can have no whitespae or multiple spaces between text fragments if we want to. We can at any time switch on smart whitespace handling by ommitting the smart\_ws=False argument, or by explicitly setting smart\_ws=True.
