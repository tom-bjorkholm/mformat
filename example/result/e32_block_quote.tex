\chapter{Block Quote Example}

Block quotes are used to highlight quoted text or to draw attention to important information.

\section{Simple Block Quote}

\begin{quote}
This is a simple block quote. Block quotes are visually distinguished from regular text with indentation and/or styling depending on the output format.
\end{quote}

\section{Block Quote with Formatting}

\begin{quote}
Block quotes can have\textbf{ bold} and\textit{ italic} text just like paragraphs.
\end{quote}

\section{Block Quote with URL}

\begin{quote}
For more information, visit \penalty0\href{http://example.com}{Example Website}\penalty0 for details.
\end{quote}

\section{Block Quote with Code}

\begin{quote}
The function \texttt{new\_block\_quote()} starts a new block quote, and \texttt{add\_text()} adds more text to it.
\end{quote}

Block quotes cannot be nested. Starting a new block quote while inside one ends the current quote and starts a fresh one.
