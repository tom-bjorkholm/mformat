\chapter{URL in numbered point list example}
\begin{enumerate}
\item A numbered point with a URL: \href{https://bitbucket.org/tom-bjorkholm/mformat/src/master/example/src}{This URL link to the examples.}
\item A numbered point with a bold URL: \textbf{\href{https://bitbucket.org/tom-bjorkholm/mformat/src/master/example/src}{This bold URL link to the example source code.}}
\item And with an italic URL: \textit{\href{https://bitbucket.org/tom-bjorkholm/mformat/src/master/example/result}{This italic URL link to the examples result.}}
\item And with an italic and bold URL: \textit{\textbf{\href{https://bitbucket.org/tom-bjorkholm/mformat/src/master/example/src}{This italic and bold URL link to the examples.}}}
\end{enumerate}
