With new\_paragraph we can start a paragraph. With add\_text we can add text to the paragraph. As described in the example file e01\_paragraph.py.

With new\_paragraph we can start a second paragraph. With add\_text we can add text to the second paragraph just as we did with the first paragraph.

(As this example does not have a heading the generated markdown file will not have a heading. If markdownlint is used on the generated markdown file it will report an error for the missing heading.)
