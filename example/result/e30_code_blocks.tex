\chapter{Code blocks example}

This is a normal paragraph with some text. Paragraphs are not useable for showing code as the text is usually shown in variable width fonts, and line wrapping is not easy to control. Code blocks on the other hand are designed to show code in a monospace font, and line wrapping is easy to control.

Code blocks are written using the \texttt{write\_code\_block()} method. The function names mentioned in this paragraph are written using the \texttt{add\_code\_in\_text()} method.

\begin{verbatim}

def hello_world(i: int) -> int:
    print("Hello, World!")
    if i > 0:
        print("i is positive")
    elif i < 0:
        print("i is negative")
    else:
        print("i is zero")
    print("This is another line of code.")
    return 42

\end{verbatim}
