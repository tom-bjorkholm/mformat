\chapter{Custom Format Example}

This example demonstrates how to create and use a custom format in the mformat framework. The same code can be used to write to different formats, just by changing the format name.

\section{Features}
\begin{itemize}
\item Easy to extend the framework
\item Supports most standard document elements
\item Format-agnostic API
\end{itemize}
\begin{enumerate}
\item First step: Create a format class
\item Second step: Implement required methods
\item Third step: Register the format
\end{enumerate}
